# Pokemon Team Builder

🔗 **[Live Demo]https://cocoayashi.github.io/pokemon-team-builder/**

A side project I built to practice Angular. You can put together a 6 Pokemon team and see how well the team holds up defensively. You will see what types it's weak to, what types it resists, and what types you have no defense against.

## What it does

- Search for any Pokemon and add them to a 6 slot team
- See each Pokemon's individual weaknesses and resistances on their card
- Get a breakdown of the whole team's defensive coverage
- See which types the team has no defense against
- Get type recommendations to fill those gaps, with the option to browse Pokemon of that type
- Dark/light mode toggle

## Built with

- Angular 21
- Angular Material
- PokéAPI
- RxJS

## Running locally

```bash
git clone https://github.com/cocoayashi/pokemon-team-builder.git
cd pokemon-team-builder
npm install
ng serve
```
