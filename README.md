# Pokemon Team Builder

🔗 **[Live Demo](https://cocoayashi.github.io/pokemon-team-builder/)**

A side project I built to learn the Angular framework. You can put together a 6 Pokemon team and see how well the team holds up defensively. You will see what types your team is weak to, what types it resists, and what types your team has no defense against.

## What it does

- Search for any Pokemon and add them to a 6 slot team
- Filter Pokemon by which game they're available in
- See each Pokemon's individual weaknesses and resistances on their card
- Get a summary of the whole team's defensive coverage
- See which types the team has no defense against
- Get type recommendations for protection against the most types, click to browse Pokemon of that type
- Dark/light mode toggle, defaults to system's preference

## Built with

- Angular 21.2.5
- Angular Material
- PokéAPI
- RxJS

## To run the app locally

```bash
git clone https://github.com/cocoayashi/pokemon-team-builder.git
cd pokemon-team-builder
npm install
ng serve
```
